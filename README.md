# EDA-on-Sleep-Data
# Exploratory Data Analysis on Sleep Data

## Project Overview
This project performs exploratory data analysis (EDA) on a sleep dataset to understand sleep patterns and factors affecting sleep quality.

## Dataset
The dataset contains information related to sleep duration, sleep quality, physical activity, stress levels, and other health indicators.

## Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data cleaning and preprocessing
- Missing value analysis
- Univariate analysis
- Bivariate analysis
- Correlation analysis
- Data visualization

## Key Findings
- Identified relationships between sleep duration and health factors.
- Analyzed the impact of lifestyle variables on sleep quality.
- Visualized trends and patterns in the dataset.

## How to Run
1. Download the repository.
2. Install the required libraries:
   ```
   pip install pandas numpy matplotlib seaborn
   ```
3. Open `eda-on-sleep-data.ipynb` in Jupyter Notebook or VS Code and run all cells.
